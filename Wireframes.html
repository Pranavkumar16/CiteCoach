🎯 Implementation Notes

  • Logo: Kinetic gradient bubble with "C" cutout and eye accent. Use SVG from uploaded file.
  • Color Scheme: Indigo-purple gradient (#6366f1 → #a855f7), white, slate grays
  • Setup Flow: Simple, non-technical. "Download Later" option available.
  • Chat Interface: Gradient message bubbles, tappable citation badges, speaker button on every answer
  • Voice: Platform native (free). Full-screen gradient overlay with live transcript.
  • Citations: ALWAYS present. Tappable. Navigate to PDF with highlighting.
  • Settings: Simplified. Offline mode locked ON. Low-power toggle. Voice speed control.
  • Navigation: 2 bottom tabs only (Library, Settings). No bookmarks tab.
  • PDF Reader: Minimal. Just chat FAB button. Clean page navigation.

🚫 NOT in V1 (Removed)

  • ❌ Bookmarks tab and features
  • ❌ Flashcards (any kind)
  • ❌ Auto-summaries during import
  • ❌ Cross-document search
  • ❌ OCR for scanned PDFs
  • ❌ Cloud storage options
  • ❌ Performance tier selector UI
  • ❌ Study sessions or spaced repetition
